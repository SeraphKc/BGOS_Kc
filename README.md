# AI-assistant

Interactive desktop application for Electron + React

## Fast start

1. **Install the dependencies:**
   ```bash
   npm install
   ```

2. **Run the application in development mode:**
   ```bash
   npm run start
   ```
   This will launch Electron with a hot reboot of the React interface.

   Either just 
    ```bash
    npm start
    ```
3. **Building the app:**
   ```bash
   npm run build
   ```
   

## Requirements
- Node.js >= 16
- npm >= 7

## Stack
- Electron
- React
- TypeScript
- Webpack

## Structure
- `src/` — application source code
- `src/components/` — React-components
- `src/services/` — services and utils

## Notes
- To run, make sure that all dependencies are installed. (`npm install`).
